## 使用过程总结

**第一阶段：** 使用cursor、Trae这类编程IDE

**第二阶段：** 使用claude code + mcp 实现快速开发和快速的测试，但开发出来的项目最后都会成为一堆屎山

**第三阶段：** 使用claude code + skills 加上各种superpower、write with file、TDD workflow、anthropic官方等等skills。编写代码的规范确实提升了很多，但还是受限于开发人员个人的需求理解能力、需求表达能力还有架构规范能力等因素。最开始看上去很好开局的一个项目，最终还是被写成了一个屎山。

**第四阶段：** 尝试学习harness engineering，并选取一个简单项目进行开发学习。
