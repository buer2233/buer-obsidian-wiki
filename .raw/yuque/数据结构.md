## 数组（Array）

数组是一种线性数据结构，用连续的内存空间存储相同类型的数据。

### 特点

- 随机访问：O(1) 时间复杂度访问任意元素
- 插入/删除：O(n) 时间复杂度（需要移动元素）
- 大小固定（静态数组）或可动态扩展（动态数组）

### Python实现

```python
# Python列表本质上是动态数组
arr = [1, 2, 3, 4, 5]

# 访问元素 - O(1)
print(arr[0])  # 1

# 末尾添加 - O(1)
arr.append(6)

# 指定位置插入 - O(n)
arr.insert(2, 10)

# 删除元素 - O(n)
arr.remove(3)

# 查找元素 - O(n)
index = arr.index(4)
```

### 常见操作时间复杂度

| 操作 | 时间复杂度 |
|------|-----------|
| 访问 | O(1) |
| 搜索 | O(n) |
| 插入（末尾） | O(1) |
| 插入（指定位置） | O(n) |
| 删除（末尾） | O(1) |
| 删除（指定位置） | O(n) |

## 队列（Queue）

队列是一种先进先出（FIFO）的线性数据结构。

### 特点

- 元素从队尾入队（enqueue）
- 元素从队头出队（dequeue）
- 先进先出

### Python实现

```python
from collections import deque

# 使用deque实现队列
queue = deque()

# 入队 - O(1)
queue.append(1)
queue.append(2)
queue.append(3)

# 出队 - O(1)
item = queue.popleft()  # 1

# 查看队头
front = queue[0]

# 判断是否为空
is_empty = len(queue) == 0
```

### 使用列表实现队列（不推荐）

```python
queue = []

# 入队 - O(1)
queue.append(1)

# 出队 - O(n)（需要移动所有元素）
item = queue.pop(0)
```

### 队列的应用

- 任务调度
- 消息队列
- 广度优先搜索（BFS）
- 缓冲区

## 堆栈（Stack）

堆栈是一种后进先出（LIFO）的线性数据结构。

### 特点

- 元素从栈顶入栈（push）
- 元素从栈顶出栈（pop）
- 后进先出

### Python实现

```python
# 使用列表实现堆栈
stack = []

# 入栈 - O(1)
stack.append(1)
stack.append(2)
stack.append(3)

# 出栈 - O(1)
item = stack.pop()  # 3

# 查看栈顶
top = stack[-1]

# 判断是否为空
is_empty = len(stack) == 0
```

### 常见操作时间复杂度

| 操作 | 时间复杂度 |
|------|-----------|
| 入栈（push） | O(1) |
| 出栈（pop） | O(1) |
| 查看栈顶（peek） | O(1) |
| 搜索 | O(n) |

### 堆栈的应用

- 函数调用栈
- 表达式求值
- 括号匹配
- 深度优先搜索（DFS）
- 浏览器的前进/后退功能

## 链表（Linked List）

链表是一种线性数据结构，通过指针将各个节点串联起来。

### 特点

- 不需要连续的内存空间
- 插入/删除操作效率高（只需修改指针）
- 不支持随机访问（需要从头遍历）

### 单链表实现

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class LinkedList:
    def __init__(self):
        self.head = None

    # 头部插入 - O(1)
    def prepend(self, val):
        new_node = ListNode(val)
        new_node.next = self.head
        self.head = new_node

    # 尾部插入 - O(n)
    def append(self, val):
        new_node = ListNode(val)
        if not self.head:
            self.head = new_node
            return
        current = self.head
        while current.next:
            current = current.next
        current.next = new_node

    # 删除节点 - O(n)
    def delete(self, val):
        if not self.head:
            return
        if self.head.val == val:
            self.head = self.head.next
            return
        current = self.head
        while current.next:
            if current.next.val == val:
                current.next = current.next.next
                return
            current = current.next

    # 查找节点 - O(n)
    def search(self, val):
        current = self.head
        while current:
            if current.val == val:
                return True
            current = current.next
        return False

    # 遍历链表
    def display(self):
        elements = []
        current = self.head
        while current:
            elements.append(current.val)
            current = current.next
        return elements
```

### 双向链表实现

```python
class DoublyListNode:
    def __init__(self, val=0, prev=None, next=None):
        self.val = val
        self.prev = prev
        self.next = next

class DoublyLinkedList:
    def __init__(self):
        self.head = None
        self.tail = None

    def append(self, val):
        new_node = DoublyListNode(val)
        if not self.head:
            self.head = new_node
            self.tail = new_node
        else:
            new_node.prev = self.tail
            self.tail.next = new_node
            self.tail = new_node
```

## 树（Tree）

树是一种非线性的层级数据结构，由节点和边组成，模拟具有层级关系的数据。树是本节的入门概念，下面会依次展开**二叉树**和**多叉树**。

### 基本术语

- **根节点（root）**：树的最顶层节点，没有父节点
- **父节点 / 子节点**：直接相连的上/下一层节点
- **叶子节点（leaf）**：没有子节点的节点
- **深度（depth）**：从根到该节点的边数（根的深度为 0）
- **高度（height）**：从该节点到最远叶子节点的边数
- **度（degree）**：节点的子节点个数

## 二叉树（Binary Tree）

二叉树是每个节点最多有两个子节点（左子树、右子树）的树结构。

### 特点

- 每个节点最多有两个子节点
- 第 i 层最多有 2^i 个节点（根为第 0 层）
- 深度为 h 的二叉树最多有 2^(h+1) - 1 个节点
- 是后续 BST、堆、平衡树的基础

### 二叉树的分类

| 类型 | 说明 |
|------|------|
| 满二叉树（Full） | 每个节点要么 0 个子节点，要么 2 个子节点 |
| 完全二叉树（Complete） | 除最后一层外全满，且最后一层节点从左到右连续 |
| 完美二叉树（Perfect） | 所有层都满，叶子都在同一层 |
| 平衡二叉树（Balanced） | 任意节点的左右子树高度差不超过 1 |

### 节点定义与遍历

```python
class TreeNode:
    def __init__(self, val=0, left=None, right=None):
        self.val = val
        self.left = left
        self.right = right

# 构造一棵示例树:
#       1
#      / \
#     2   3
#    / \
#   4   5
root = TreeNode(1)
root.left = TreeNode(2)
root.right = TreeNode(3)
root.left.left = TreeNode(4)
root.left.right = TreeNode(5)


# ========== 递归遍历（DFS）==========
# 前序遍历：根 -> 左 -> 右
def preorder(node, result=None):
    if result is None:
        result = []
    if node:
        result.append(node.val)        # 先访问根
        preorder(node.left, result)
        preorder(node.right, result)
    return result

# 中序遍历：左 -> 根 -> 右（对BST得到有序序列）
def inorder(node, result=None):
    if result is None:
        result = []
    if node:
        inorder(node.left, result)
        result.append(node.val)
        inorder(node.right, result)
    return result

# 后序遍历：左 -> 右 -> 根（常用于释放节点）
def postorder(node, result=None):
    if result is None:
        result = []
    if node:
        postorder(node.left, result)
        postorder(node.right, result)
        result.append(node.val)
    return result

print(preorder(root))   # [1, 2, 4, 5, 3]
print(inorder(root))    # [4, 2, 5, 1, 3]
print(postorder(root))  # [4, 5, 2, 3, 1]


# ========== 层序遍历（BFS）==========
from collections import deque

def level_order(root):
    if not root:
        return []
    result = []
    queue = deque([root])
    while queue:
        node = queue.popleft()
        result.append(node.val)
        if node.left:
            queue.append(node.left)
        if node.right:
            queue.append(node.right)
    return result

print(level_order(root))  # [1, 2, 3, 4, 5]
```

### 迭代遍历（用栈模拟递归）

```python
# 前序遍历迭代版 - O(n) 时间，O(h) 空间
def preorder_iterative(root):
    if not root:
        return []
    result, stack = [], [root]
    while stack:
        node = stack.pop()
        result.append(node.val)
        # 先压右再压左，保证弹出顺序是 左 -> 右
        if node.right:
            stack.append(node.right)
        if node.left:
            stack.append(node.left)
    return result

# 中序遍历迭代版
def inorder_iterative(root):
    result, stack = [], []
    current = root
    while current or stack:
        # 一路向左压栈
        while current:
            stack.append(current)
            current = current.left
        current = stack.pop()
        result.append(current.val)
        current = current.right
    return result
```

# 判断二叉树是否镜像对称（根的左右子树互为镜像）

#             2     2

def is_symmetric(root):

## 二叉搜索树（BST）

BST 是一种特殊的二叉树：左子树所有值 < 根 < 右子树所有值，且左右子树也分别是 BST。

### 特点

- 中序遍历得到有序序列
- 查找/插入/删除平均 O(log n)（树平衡时）
- 退化成链表时最坏 O(n)

### Python实现（插入与查找）

```python
class BSTNode:
    def __init__(self, val):
        self.val = val
        self.left = None
        self.right = None

class BST:
    def __init__(self):
        self.root = None

    # 插入 - 平均 O(log n)，最坏 O(n)
    def insert(self, val):
        if not self.root:
            self.root = BSTNode(val)
            return
        current = self.root
        while True:
            if val < current.val:
                if current.left:
                    current = current.left
                else:
                    current.left = BSTNode(val)
                    return
            else:
                if current.right:
                    current = current.right
                else:
                    current.right = BSTNode(val)
                    return

    # 查找 - O(h)
    def search(self, val):
        current = self.root
        while current:
            if val == current.val:
                return True
            current = current.left if val < current.val else current.right
        return False
```

> **延伸：** BST 的性能完全依赖树的形状。有序插入会让它退化成链表，因此工程中常用**平衡树**（AVL、红黑树）在插入/删除后自动旋转保持平衡，把最坏情况也压到 O(log n)。Python 标准库没有内置平衡树，但 `bisect` 模块用有序数组 + 二分提供了类似的有序维护能力。

## 多叉树（N-ary Tree）

多叉树是每个节点可以有任意数量子节点的树，不受"最多两个"的限制。

### 特点

- 每个节点的子节点数量不限（0 个到任意个）
- 用 children 列表存储子节点，而非固定 left/right 指针
- 文件系统目录、组织架构、XML/HTML DOM 都是天然的多叉树
- 当节点度为固定常数 m 时称为 m 叉树

### 节点定义与遍历

```python
class NaryTreeNode:
    def __init__(self, val=None):
        self.val = val
        self.children = []  # 子节点列表，数量任意

# 构造示例多叉树:
#         1
#      /  |  \
#     3   2   4
#    / \
#   5   6
root = NaryTreeNode(1)
node3 = NaryTreeNode(3)
root.children = [node3, NaryTreeNode(2), NaryTreeNode(4)]
node3.children = [NaryTreeNode(5), NaryTreeNode(6)]


# 前序遍历（根 -> 子节点从左到右）
def nary_preorder(node, result=None):
    if result is None:
        result = []
    if node:
        result.append(node.val)
        for child in node.children:
            nary_preorder(child, result)
    return result

# 后序遍历（子节点从左到右 -> 根）
def nary_postorder(node, result=None):
    if result is None:
        result = []
    if node:
        for child in node.children:
            nary_postorder(child, result)
        result.append(node.val)
    return result

# 层序遍历（BFS）
def nary_level_order(root):
    if not root:
        return []
    result, queue = [], deque([root])
    while queue:
        node = queue.popleft()
        result.append(node.val)
        # 与二叉树唯一不同：把所有子节点依次入队
        queue.extend(node.children)
    return result

print(nary_preorder(root))   # [1, 3, 5, 6, 2, 4]
print(nary_postorder(root))  # [5, 6, 3, 2, 4, 1]
print(nary_level_order(root))  # [1, 3, 2, 4, 5, 6]
```

### 二叉树 vs 多叉树对比

| 维度 | 二叉树 | 多叉树 |
|------|--------|--------|
| 子节点数 | 最多 2 个（left/right） | 任意个（children 列表） |
| 存储方式 | 两个固定指针 | 动态列表/数组 |
| 遍历 | 前/中/后 + 层序 | 前序/后序/层序（无"中序"） |
| 典型场景 | 查找结构、表达式树 | 文件系统、组织架构、DOM |
| 查找效率 | 平衡时 O(log n) | 取决于扇出和结构 |

### 常见操作时间复杂度

| 操作 | 时间复杂度 | 说明 |
|------|-----------|------|
| 遍历 | O(n) | 访问每个节点一次 |
| 访问/搜索 | O(n) | 通常需遍历，无序则无捷径 |
| 插入 | O(1) | 挂到指定父节点的 children 末尾 |
| 删除子树 | O(1) | 断开指针即可（不含释放） |
| 空间（递归） | O(h) | h 为树高，递归栈深度 |

### 多叉树的应用

- 文件系统目录结构（一个目录下多个文件/子目录）
- 组织架构图（一个上级多个下属）
- XML / HTML 的 DOM 树
- 前端组件树（React/Vue 的虚拟 DOM）
- 字典树（Trie，多叉树特化用于字符串前缀检索）

## 哈希表（Hash Table）

哈希表是一种通过 **key -> value** 映射来存储数据的数据结构。它用哈希函数把 key 转换成数组下标，从而实现平均 O(1) 的查找、插入和删除。

### 特点

- 通过哈希函数定位数据存储位置
- 平均查找、插入、删除都是 O(1)
- 可能发生哈希冲突，需要用链地址法、开放寻址法等方式解决
- Python 中的 `dict` 和 `set` 都基于哈希表思想

### Python实现

```python
# dict：key -> value 映射
scores = {}

# 插入 / 更新 - 平均 O(1)
scores["Tom"] = 95
scores["Jerry"] = 88

# 查询 - 平均 O(1)
tom_score = scores["Tom"]

# 判断 key 是否存在 - 平均 O(1)
exists = "Tom" in scores

# 删除 - 平均 O(1)
del scores["Jerry"]

# set：只关心元素是否存在
seen = set()
seen.add("apple")
is_seen = "apple" in seen
```

### 哈希冲突

不同的 key 可能经过哈希函数后得到相同的位置，这就是哈希冲突。

| 解决方式 | 思路 |
|----------|------|
| 链地址法 | 每个桶后面挂一个链表或其他结构，冲突元素放到同一个桶里 |
| 开放寻址法 | 冲突后继续探测下一个可用位置 |
| 扩容再哈希 | 元素过多时扩大数组容量，降低冲突概率 |

### 常见操作时间复杂度

| 操作 | 平均时间复杂度 | 最坏时间复杂度 |
|------|----------------|----------------|
| 查找 | O(1) | O(n) |
| 插入 | O(1) | O(n) |
| 删除 | O(1) | O(n) |
| 遍历 | O(n) | O(n) |

### 哈希表的应用

- 快速查找与去重
- 缓存系统
- 统计词频、计数器
- 两数之和等查找类算法题
- 数据库 Hash 索引

## 堆（Heap）

堆是一种满足堆序性质的完全二叉树，常用数组实现。Python 标准库 `heapq` 默认实现的是**小顶堆**：父节点的值总是不大于子节点。

### 特点

- 逻辑上是完全二叉树，物理上常用数组存储
- 小顶堆：堆顶是最小值
- 大顶堆：堆顶是最大值
- 插入和删除堆顶都需要通过上浮 / 下沉维持堆序

### 数组下标关系

假设当前节点下标为 `i`：

| 关系 | 下标 |
|------|------|
| 父节点 | `(i - 1) // 2` |
| 左孩子 | `2 * i + 1` |
| 右孩子 | `2 * i + 2` |

### Python实现

```python
import heapq

# Python heapq 默认是小顶堆
heap = []

# 入堆 - O(log n)
heapq.heappush(heap, 5)
heapq.heappush(heap, 2)
heapq.heappush(heap, 8)

# 查看堆顶 - O(1)
top = heap[0]  # 2

# 弹出堆顶 - O(log n)
min_value = heapq.heappop(heap)

# 将列表原地堆化 - O(n)
nums = [5, 2, 8, 1]
heapq.heapify(nums)
```

### 大顶堆写法

```python
import heapq

nums = [5, 2, 8, 1]
max_heap = []

for num in nums:
    heapq.heappush(max_heap, -num)

max_value = -heapq.heappop(max_heap)  # 8
```

## 图（Graph）

图是一种由**顶点（Vertex）**和**边（Edge）**组成的非线性数据结构，用来表示对象之间的关系。

### 基本概念

- **顶点（Vertex）**：图中的节点
- **边（Edge）**：顶点之间的连接关系
- **有向图**：边有方向，例如 A -> B
- **无向图**：边没有方向，例如 A -- B
- **带权图**：边带有权重，例如距离、成本、耗时
- **度（Degree）**：与某个顶点相连的边数

### 存储方式

| 存储方式 | 说明 | 适合场景 |
|----------|------|----------|
| 邻接表 | 每个顶点保存相邻顶点列表 | 稀疏图，工程中更常用 |
| 邻接矩阵 | 用二维数组表示顶点连接关系 | 稠密图，适合快速判断两点是否相连 |

### 邻接表实现

```python
# 无向图邻接表
graph = {
    "A": ["B", "C"],
    "B": ["A", "D"],
    "C": ["A", "D"],
    "D": ["B", "C"]
}

# 添加边
def add_edge(graph, u, v):
    graph.setdefault(u, []).append(v)
    graph.setdefault(v, []).append(u)

add_edge(graph, "A", "D")
```

### BFS遍历

```python
from collections import deque

def bfs(graph, start):
    visited = set([start])
    queue = deque([start])
    result = []

    while queue:
        node = queue.popleft()
        result.append(node)
        for neighbor in graph.get(node, []):
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)

    return result
```

### DFS遍历

```python
def dfs(graph, start, visited=None, result=None):
    if visited is None:
        visited = set()
    if result is None:
        result = []

    visited.add(start)
    result.append(start)

    for neighbor in graph.get(start, []):
        if neighbor not in visited:
            dfs(graph, neighbor, visited, result)

    return result
```

### 常见操作时间复杂度

设顶点数为 V，边数为 E：

| 操作 | 邻接表 | 邻接矩阵 |
|------|--------|----------|
| 存储空间 | O(V + E) | O(V²) |
| 判断两点是否相连 | O(degree) | O(1) |
| 遍历所有邻居 | O(degree) | O(V) |
| BFS / DFS | O(V + E) | O(V²) |

### 图的应用

- 社交网络关系
- 地图路径规划
- 依赖关系与拓扑排序
- 网络路由
- 知识图谱
- 推荐系统

## B 树 / B+ 树（B-Tree / B+ Tree）

B 树和 B+ 树都是多路平衡查找树，常用于数据库和文件系统索引。它们的核心目标是减少磁盘 IO：每个节点存放多个 key，让一次读取能获得更多索引信息。

### B 树特点

- 一个节点可以存多个 key 和多个子节点
- 所有叶子节点大致在同一层，树高度较低
- 数据可以存储在内部节点，也可以存储在叶子节点
- 查找、插入、删除都能保持平衡

### B+ 树特点

- 非叶子节点主要存 key，用于索引导航
- 真正的数据通常都在叶子节点
- 叶子节点之间通过链表相连，适合范围查询
- 数据库索引中更常见的是 B+ 树

### B 树 vs B+ 树

| 维度 | B 树 | B+ 树 |
|------|------|-------|
| 数据位置 | 内部节点和叶子节点都可存数据 | 数据主要存叶子节点 |
| 范围查询 | 需要中序遍历树 | 叶子链表顺序扫描更高效 |
| 查询稳定性 | 命中内部节点可提前结束 | 通常都走到叶子节点 |
| 典型应用 | 文件系统、部分索引结构 | MySQL InnoDB 索引等数据库场景 |

### 简化节点结构

```python
class BTreeNode:
    def __init__(self, leaf=False):
        self.keys = []      # 节点中的多个 key
        self.children = []  # 多个子节点
        self.leaf = leaf


class BPlusTreeLeaf:
    def __init__(self):
        self.keys = []
        self.values = []
        self.next = None    # 指向下一个叶子节点，方便范围查询
```

### B+ 树范围查询示意

```python
def range_scan(leaf, start, end):
    result = []
    current = leaf

    while current:
        for key, value in zip(current.keys, current.values):
            if start <= key <= end:
                result.append(value)
            if key > end:
                return result
        current = current.next

    return result
```

### 常见操作时间复杂度

| 操作 | 时间复杂度 |
|------|-----------|
| 查找 | O(log n) |
| 插入 | O(log n) |
| 删除 | O(log n) |
| 范围查询 | O(log n + k)，k 为结果数量 |

### B 树 / B+ 树的应用

- 数据库索引
- 文件系统索引
- 大规模有序数据存储
- 范围查询、排序查询
- 减少磁盘 IO 的索引结构
