## 数组（Array）

数组是一种线性数据结构，用连续的内存空间存储相同类型的数据。

### 特点

- 随机访问：O(1) 时间复杂度访问任意元素
- 插入/删除：O(n) 时间复杂度（需要移动元素）
- 大小固定（静态数组）或可动态扩展（动态数组）

### Python实现

```python
# Python列表本质上是动态数组
arr = [1, 2, 3, 4, 5]

# 访问元素 - O(1)
print(arr[0])  # 1

# 末尾添加 - O(1)
arr.append(6)

# 指定位置插入 - O(n)
arr.insert(2, 10)

# 删除元素 - O(n)
arr.remove(3)

# 查找元素 - O(n)
index = arr.index(4)
```

### 常见操作时间复杂度

| 操作 | 时间复杂度 |
|------|-----------|
| 访问 | O(1) |
| 搜索 | O(n) |
| 插入（末尾） | O(1) |
| 插入（指定位置） | O(n) |
| 删除（末尾） | O(1) |
| 删除（指定位置） | O(n) |

## 队列（Queue）

队列是一种先进先出（FIFO）的线性数据结构。

### 特点

- 元素从队尾入队（enqueue）
- 元素从队头出队（dequeue）
- 先进先出

### Python实现

```python
from collections import deque

# 使用deque实现队列
queue = deque()

# 入队 - O(1)
queue.append(1)
queue.append(2)
queue.append(3)

# 出队 - O(1)
item = queue.popleft()  # 1

# 查看队头
front = queue[0]

# 判断是否为空
is_empty = len(queue) == 0
```

### 使用列表实现队列（不推荐）

```python
queue = []

# 入队 - O(1)
queue.append(1)

# 出队 - O(n)（需要移动所有元素）
item = queue.pop(0)
```

### 队列的应用

- 任务调度
- 消息队列
- 广度优先搜索（BFS）
- 缓冲区

## 堆栈（Stack）

堆栈是一种后进先出（LIFO）的线性数据结构。

### 特点

- 元素从栈顶入栈（push）
- 元素从栈顶出栈（pop）
- 后进先出

### Python实现

```python
# 使用列表实现堆栈
stack = []

# 入栈 - O(1)
stack.append(1)
stack.append(2)
stack.append(3)

# 出栈 - O(1)
item = stack.pop()  # 3

# 查看栈顶
top = stack[-1]

# 判断是否为空
is_empty = len(stack) == 0
```

### 常见操作时间复杂度

| 操作 | 时间复杂度 |
|------|-----------|
| 入栈（push） | O(1) |
| 出栈（pop） | O(1) |
| 查看栈顶（peek） | O(1) |
| 搜索 | O(n) |

### 堆栈的应用

- 函数调用栈
- 表达式求值
- 括号匹配
- 深度优先搜索（DFS）
- 浏览器的前进/后退功能

## 链表（Linked List）

链表是一种线性数据结构，通过指针将各个节点串联起来。

### 特点

- 不需要连续的内存空间
- 插入/删除操作效率高（只需修改指针）
- 不支持随机访问（需要从头遍历）

### 单链表实现

```python
class ListNode:
    def __init__(self, val=0, next=None):
        self.val = val
        self.next = next

class LinkedList:
    def __init__(self):
        self.head = None

    # 头部插入 - O(1)
    def prepend(self, val):
        new_node = ListNode(val)
        new_node.next = self.head
        self.head = new_node

    # 尾部插入 - O(n)
    def append(self, val):
        new_node = ListNode(val)
        if not self.head:
            self.head = new_node
            return
        current = self.head
        while current.next:
            current = current.next
        current.next = new_node

    # 删除节点 - O(n)
    def delete(self, val):
        if not self.head:
            return
        if self.head.val == val:
            self.head = self.head.next
            return
        current = self.head
        while current.next:
            if current.next.val == val:
                current.next = current.next.next
                return
            current = current.next

    # 查找节点 - O(n)
    def search(self, val):
        current = self.head
        while current:
            if current.val == val:
                return True
            current = current.next
        return False

    # 遍历链表
    def display(self):
        elements = []
        current = self.head
        while current:
            elements.append(current.val)
            current = current.next
        return elements
```

### 双向链表实现

```python
class DoublyListNode:
    def __init__(self, val=0, prev=None, next=None):
        self.val = val
        self.prev = prev
        self.next = next

class DoublyLinkedList:
    def __init__(self):
        self.head = None
        self.tail = None

    def append(self, val):
        new_node = DoublyListNode(val)
        if not self.head:
            self.head = new_node
            self.tail = new_node
        else:
            new_node.prev = self.tail
            self.tail.next = new_node
            self.tail = new_node
```

### 常见操作时间复杂度

| 操作 | 单链表 | 双向链表 |
|------|-------|---------|
| 头部插入 | O(1) | O(1) |
| 尾部插入 | O(n) | O(1) |
| 中间插入 | O(n) | O(n) |
| 删除 | O(n) | O(n) |
| 搜索 | O(n) | O(n) |

### 链表的应用

- 实现其他数据结构（栈、队列）
- LRU缓存
- 文件系统
- 音乐播放列表
- 浏览器历史记录
