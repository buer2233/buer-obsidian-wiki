# Python高级总结

## 面向对象基础

### 面向过程

#### 实现一个事物或是完成一个功能, 考虑的是一步, 一步依次达到, 注重过程
- C语言

### 面向对象

#### 实现一个事物或是完成一个功能, 不考虑过程, 只关心结果
- C#,C++,Python,PHP,Java

### 类

#### 描述一类事物,包含事物的属性和方法.

### 对象

#### 类的实例化.具体的一个事物,包含这个事物的属性和方法

## 类的定义

### 创建类

#### class 类名(父类名):  # 类名使用大驼峰命名
	'''多行注释对类进行说明'''
	类属性 = 值  # 类和对象均可使用
	def __init__(self, 实例属性1,实例属性2, ..,实例属性N):
		self.实例属性1 = 实例属性1
		self.实例属性2 = 实例属性2
		......
		self.实例属性N = 实例属性N
	def 实例方法(self):
		实例方法代码
	@classmethod
	def 类方法(cls):
		类方法代码  # 一般会使用类属性
	@staticmethod
	def 静态方法():
		静态方法代码

### 通过类实例化对象

#### 对象名 = 类名(形参1, 形参2, ...,形参N)

### 操作对象

#### 属性的操作
- 新增属性
- 对象名.不存在的属性名 = 值
- 删除属性
- del 对象名.属性名
- 只能修改对象的属性,无法修改类的属性
- 修改属性
- 对象名.属性名 = 新值
- 查询属性
- print(对象名.属性名)

#### 方法的操作
- 对象名.方法名()

## 类的加强

### self说明

#### 实例化的对象
- self.方法名
- 类中有多个方法, 其中一个方法要调用类中其它的方法

#### 哪个对象调用就是哪个对象
- self. 属性名/ self.方法名
- 哪个对象调用就是哪个对象的属性/ 哪个对象的方法

### 私有成员

#### 成员
- 类中的属性和方法统称为成员

#### 定义方法
- 在成员前加两个下划线的成员就是私有成员: __成员名

#### 特点
- 私有成员仅能由类自己使用

#### 作用
- 更好的保护属性的安全,即不能随意修改
- 面向对象的编程思维只关注,对象功能完成的结果,不关注内部的实现过程,设计私有成员可以提高代码的封装性,隐蔽内部实现,仅保留开发人员需要的功能.

#### 非法访问私有属性的方法
- dir()
- 查看对象所有的属性和方法,会显示被改名的私有属性
- _类名__属性名/__方法名()

### 类属性

#### 定义在类里的函数外的属性

#### 可以同时被类和对象使用

### 实例属性

#### 特点
- 只能被具体的实例对象使用
- 实例对象直接属性是独立的

#### 创建方式
- 1. __init__(self): 初始化实例属性
- 2. 创建对象后新增属性(修改)

### 类方法

#### 语法
- @classmethod
def 类方法(cls):
  类方法代码

#### 特点
- 类和对象均可使用
- 一般会使用类属性

### 静态方法

#### 语法
- @staticmethod
def 静态方法():
  静态方法代码

#### 特点
- 类和对象均可使用
- 一般为通用的方法,但又与类的特点关系性不大.
例如: 时间方法等

### 继承

#### 定义
- 发生在两个类之间, 子类可以继承父类的所有方法和属性(私有成员除外)

#### 分类
- 单继承
- 一个子类继承一个父类
- 多继承
- 当一个子类需要同时继承两个没有直接关系的父类时使用
class(父类1, 父类2, ...)
- 多层继承
- C类继承B类, B类继承A类
- C类可以同时使用B类和A类的属性和方法

#### 重写
- 定义
- 当子类中出现了和父类一样的属性名或方法名时,子类就对父类进行了重写
- super()方法
- 子类重写父类的属性或方法之后,强制调用父类属性或方法的函数
- super(子类名, self).父类的方法或属性

### 多态

#### 在Python中,可以在多个子类中重写父类的方法,这时所有的子类和父类虽然拥有相同名字的方法,但各自的效果不同,这就是多态

### 魔法方法

#### __new__
- 系统自带,并默认执行的方法.在创建对象的时候调用,且在__init__前
- 作用
- 创建对象,并为对象开辟内存空间
- 将创建的对象以 self 的形式返回给__init__初始化属性或实例化方法
- 单例模式
- 无论实例化多少次对象, 都只能得到一个对象, 内存地址相同
- 实现方式
- 改写 __new__ 方法

#### __del__
- 在对象在内存中被销毁时调用,用于提示这个对象被销毁了,或是程序运行结束
- 对象内存被销毁的情况
- 程序运行结束后,销毁对象的内存
- 当对象没有任何变量引用时,被销毁

#### __str__
- 自定义类和对象的显示

### isinstance(对象, 指定类型)

#### 判断对象是否是指定的类型

## 异常处理

### 完整语法

#### try:
  可能异常的代码块
except 异常类名 as 别名:
  发生异常类名对应的异常时,执行的代码块
except Exception as 别名:
  发生所有代码异常均执行的代码块
else:
  未发生异常执行的代码块
finally:
  无论有没有异常均执行的代码块  # 通常做一些清理工作,关闭文件,关闭数据库之类的.无论前面的运行情况均需要执行的代码

### 异常的定义

#### 当发生错误时,后面的代码就不在执行了,但不包含语法错误

### 分类

#### 子类
- NameError, TypeError, FileNotFindError等
- try:
  可能异常的代码块
except 异常模块名 as e:
  出现异常类名的错误时执行的代码

#### 父类
- Exception
- try:
  可能异常的代码块
except Exception as e:
  不明确错误类型时, 所有代码异常均执行的代码块

### 异常的传递

#### 报错的排查从最后一行开始

### 异常的嵌套

#### 如果一层try无法捕获异常, 可以在外层嵌套继续try

### 异常的抛出raise

#### raise()
- 抛出上一个异常

#### raise 异常类名("报错提示信息")
- 改写异常报错的提示信息

### 模块

#### 定义
- 在Python中仅包含变量/函数/类的一个py文件,就叫模块

#### 引入语法
- import 模块名
- import 模块1, 模块2

### 包

#### 引入语法

- 不能直接使用, 如果 __init__.py 中没有 __all__, 则使用会报错

### 时间日期模块

#### 时间戳
- 以英国的格林尼治天文台的时间, 从1970年1月1日 00:00:00开始, 到现在的秒数

#### 基础时间(time)
- 时间戳只能从1970年1月1日开始,到2038年1月1日(32位)
- time.time()
- 打印时间戳的秒数
- time.sleep(秒数)
- 控制程序休眠指定的秒数,再执行后面的程序

#### 高级时间(datetime)
- 时间戳从1970年1月1日开始 到 9999年 (64位)
- datetime.datetime(年, 月, 日, 时, 分, 秒)
- 实例化一个具体的时间对象,至少需要包含年,月,日
- datetime.datetime.now()
- 实例化一个当前时间的时间对象
- datetime.datetime.strftime(时间对象, 字符串时间格式(%Y-%m-%d))
- 将时间对象转换为字符串时间
- datetime.datetime.strptime(字符串时间, 时间对象格式(%Y-%m-%d))
- 将字符串时间转换为时间对象
- 转换时, 字符串时间的格式需与时间对象的格式完全一致
- datetime.timedelta(day= , hours= , weeks=)
- 时间块: 通常用于时间之间的运算, 或是运算后的结果
- 工作中常用的场景

### 随机数模块

#### import random

#### random.random()
- 随机取0~1之间的浮点数, 不包含1

#### random.randomint(a, b)
- 随机取a和b之间的整数

#### random.choice(容器)
- 随机取容器中的一个数

#### random.sample(容器, k)
- 随机从容器中取 k 个数,返回为列表

### 哈希散列模块

#### import hashlib

#### md5加密
- 将源数据转化为16进制(8\16\32位), 固定长度的加密数据
- 语法
- hashlib.md5(需要加密的源数据)
- hashlib.md5("源数据".encode()).hexdigest
- 注意
- 进行的加密的源数据必须是二进制
- 如果源数据为ASCII字符(字母,数字,部分符号)
- b"需要加密的源数据"
- 如果源数据包含非ASCII字符(中文,特殊符号等)
- "需要加密的源数据".encode()
- 加密直接得到的是一个md5对象, 如果要查看md5的值
- md5对象.hexdigest
- 特点
- 只能加密, 不能解密
- 同一个源数据, 加密得到的结果是固定不变的(由固定的哈希算法计算的结果)
- 不管源数据是否长度不同, 加密出来的加密数据长度都是一样的
- 因为长度较短, 安全性偏低

#### sha2
- 长度远远大于md5, 安全性更高

## 其它高级内容

### 拷贝

#### is 和 ==
- is
- 判断内存地址是否相同
- 内存地址相同,值一定相同
- ==
- 判断值是否相同
- 值相同,内存地址不一定相同

#### 赋值
- 赋值操作时, 等号左边相当于 key, 右边相当于 value
- 赋值的过程就是, 在key空间中,定义一个别名,具体别名指向的内存地址

#### 浅拷贝
- 仅拷贝内存地址(第一层), 重新开启空间存放
- 当拷贝的数据包含可变数据类型时, 因浅拷贝只拷贝内存地址.当可变数据变化时,源数据和浅拷贝数据都会变化

#### 深拷贝
- 使用递归复制的方式拷贝所有的值(每一层),重新开辟空间存放
- 源数据和深拷贝数据,没有关联关系,互不影响

### 名字空间

#### 定义
- 存放变量名,函数名,类名的地方.是一种字典数据类型

#### 内键命名空间
- 通过 dir(__builtins__)查看
- 保存Python自带的变量\函数名\类名

#### 全局命名空间
- globals()查看
- 保存全局变量\函数\类名

#### 局部命名空间
- locals()
- 保存局部变量\函数名\类名

#### 特点
- 不同命名空间,可以重复使用相同的变量名\函数名\类名
- 在同一命名空间,不能使用重复的名字(如果重复,后写的会覆盖前面的)

### 作用域

#### 局部变量(Local)
- 在函数内定义的变量,只能在函数内使用

#### 闭包变量(Enclosed)
- 当函数内嵌函数时,在内嵌函数的外层定义的变量称为闭包变量

#### 全局变量(Global)
- 在函数外定义的,既能在函数外也能在函数内使用的变量

#### 内置变量(Built-in)
- Python自己定义的变量

#### 查找顺序
- LEGB法则
- local局部 --> Enclosed闭包 --> Global全局 --> Built-in内置

### 正则表达式(re)

#### 定义
- 对字符串进行过滤的一种工具

#### re.search(re表达式, 字符串)
- 匹配到第一个符合的字符串,即终止过滤,并返回字符串对象
- 未匹配到符合的字符串时,返回None
- 返回的为一个正则表达式对象, 可使用: 正则表达式对象.group() 转换为字符

#### re.findall(re表达式, 字符串)
- 查找所有符合条件的字符串, 并以列表的方式返回
- 未查找到对应字符,返回空列表

#### 贪婪模式和非贪婪模式
- 贪婪模式
- 系统默认的模式,尽可能多的匹配符合条件的字符
- 非贪婪模式
- 通过 ? 号使用费贪婪模式, 匹配到符合条件的字符即停止

#### 常用re字符
- ^
- 匹配字符开头
- $
- 匹配字符结束
- .
- 匹配任意字符, 换行符除外
- []
- 表示一组字符
- \w
- 匹配包括下划线,字母,数字的字符
- \s
- 匹配任意空白字符,包括: 空格, tab, 换行符
- \d
- 匹配任意数字
- [0-9]
- 匹配任意数字
- [a-z]
- 匹配任意小写字母
- [A-Z]
- 匹配任意大写字母
- *
- 匹配任意多个字符
- +
- 匹配一个以上的任意字符
- ?
- 开始非贪婪模式

### 装饰器

#### 定义
- 在不改变原函数的基础上,为原函数增加新功能

#### 语法
- 第一步: 写装饰函数

### 迭代器和生成器

#### 容器
- 是一种把多个元素组织在一起的数据结构,容器中的元素可以逐个迭代获取,也可以使用 in, not in 进行判断

#### 可迭代对象
- 可以返回一个迭代器的对象就是迭代器对象

#### 迭代器
- 将容器中的元素进行迭代的工具
- 创建迭代器
- 迭代器 = iter(可迭代对象)
- 使用迭代器
- next(迭代器)
- 当元素被取完之后,继续使用next(),报错StopIteration

#### 生成器
- 是一种特殊的,更加优雅的迭代器.它只有当元素需要被使用时,才会调用元素,为元素开辟内存空间,使用后马上销毁.可以大大的降低内存占用,可用于操作较大的数据量.
