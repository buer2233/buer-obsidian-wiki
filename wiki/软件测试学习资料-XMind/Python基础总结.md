# Python基础

## Python介绍

### Python是一种面向对象的解释性计算机程序设计语言,由荷兰人Guido van Rossum(龟叔)于1989年发明.

### 编译性语言

#### C、C++、
- 使用专门的编译器，针对特定的平台，将高级语言源代码一次性的编译成可被该平台硬件执行的机器码，并包装成该平台所能识别的可执行性程序的格式。
- 特点:在编译型语言写的程序执行之前，需要一个专门的编译过程，把源代码编译成机器语言的文件，如exe格式的文件，以后要再运行时，直接使用编译结果即可，如直接运行exe文件。因为只需编译一次，以后运行时不需要编译，所以编译型语言执行效率高。

### 解释型语言

#### Python
- 使用专门的解释器对源程序逐行解释成特定平台的机器码并立即执行。
- 特点:解释型语言不需要事先编译，其直接将源代码解释成机器码并立即执行，所以只要某一平台提供了相应的解释器即可运行该程序。

## 标识符

### 标识符就是程序员在程序中自定义的变量名, 函数名, 类名.

### 命名规则

#### 下划线命名法
- 单词和单词之间使用下划线分割。例如：send_keys、file_not_found_error

#### 小驼峰命名法
- 小驼峰式命名法：第一个单词首字母小写，以后单词的首字母大写。例如：myName、fileNotFoundError

#### 大驼峰命名法
- 每一个单字的首字母均大写,用于类名的命名。例如：FirstName、FileNotFoundError

## 函数 def

### 定义

#### 函数是具有名字的可重复执行的代码块, 通过函数名()去调用执行

### 函数定义的三部曲

#### 1. 确定函数名, 做到见名知意, 符合标识符命名规则

#### 2. 确定函数的形式参数, 当函数中有未知数时, 就需要对应的形参

#### 3. 确定函数返回值
- 不清楚是否需要返回, 就返回
- 如果后续的代码需要用到函数的计算结果, 则需要返回
- 如果需求明确说明调用函数, 获得/获利/得到 就需要返回
- 如果调用的时候, 明确输出/打印, 就不需要返回

### 函数参数

#### 形式参数
- 写在函数名()里的参数
- 必传参数
- 定义时, 没有写默认值得参数, 就是必传参数
- 默认参数
- 定义时写了默认值的形参, 可以不用传入实参, 则使用默认值.

#### 实际参数
- 写在调用函数()里的参数
- 位置参数
- 按照与形参一一对应的位置传入实参
- 关键字参数
- 通过 形参 = 值 的形式, 传入实参, 位置不用与形参一一对应

#### 不定长参数(打包)
- 不定长的位置参数打包
- def 函数名(*args):
- 将实参传入的多个位置参数, 打包为元组
- 不定长的关键字参数打包
- def 函数名(**kwargs)
- 将实参传入的多个关键字参数, 打包为字典

#### 实际参数的拆包
- 函数名(*元组 / 列表)
- 将元组 或 元组 的所有元素拆包为多个实际位置参数
- 函数名(**字典)
- 将字典的所有键值对, 拆包为多个关键字实际参数传入

### return

#### 将数据返回函数的调用位置

#### 无返回值 none
- return后不写数据, 则返回none

#### 多处返回值
- 函数可以根据实际情况在多个位置使用return, 但只能执行一个. 因为一旦执行return, 函数就结束了

#### 返回多个值
- 函数可以一次返回多个值, 同时也需要多个变量保存返回的值

#### 退出函数(终止函数继续执行)

### pass

### 匿名函数

- 一般用做其它函数的实际参数传入其它函数

### 变量的作用域

#### 全局变量
- 定义在函数外的变量, 可在整个py文件中使用,  不能跨模块

#### 局部变量

- 将局部变量转化为全局变量, 在定义前转换

### 切片

#### 语法
- 新的列表 = 列表[起始索引 : 结束索引 : 步长]

#### 注意点1
- 左闭右开, 包含左不包含右
- 不写起始索引: 默认为0或-1, 由步长决定的方向改变
- 不写结束索引: 默认到列表结尾
- 不写步长: 默认为1
- 切片之后是得到一个新的列表, 不会修改原列表

#### 注意点2
- 不用考虑索引的正负值, 只考虑方向是否冲突就行
- 切片的结束索引超出列表的范围, 也不会报错
- 同一索引范围在不同的列表中, 对应的方向不一定一样. 因此当不清楚列表的值时, 就无法判断正负索引的方向, 此时应尽量避免混用正负索引作为开始和结束索引
