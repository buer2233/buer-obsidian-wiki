# 算法

## 1. 时间复杂度

### 什么是时间复杂度

时间复杂度表示算法执行时间随输入规模增长的变化趋势。

### 大O符号

大O符号表示算法的**上界**，即最坏情况下的时间复杂度。

### 常见时间复杂度

| 复杂度 | 名称 | 示例 |
|--------|------|------|
| O(1) | 常数 | 数组访问、哈希表查找 |
| O(log n) | 对数 | 二分查找 |
| O(n) | 线性 | 遍历数组 |
| O(n log n) | 线性对数 | 归并排序、快速排序 |
| O(n²) | 平方 | 冒泡排序、选择排序 |
| O(n³) | 立方 | 矩阵乘法 |
| O(2^n) | 指数 | 子集生成 |
| O(n!) | 阶乘 | 全排列 |

### 复杂度增长对比

```
n       O(1)    O(log n)  O(n)    O(n log n)  O(n²)
10      1       3         10      33          100
100     1       7         100     664         10000
1000    1       10        1000    9966        1000000
10000   1       13        10000   132877      100000000
```

### 常见操作复杂度

```python
# O(1) - 常数时间
def get_first(arr):
    return arr[0]

# O(log n) - 对数时间
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

# O(n) - 线性时间
def linear_search(arr, target):
    for i, num in enumerate(arr):
        if num == target:
            return i
    return -1

# O(n²) - 平方时间
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
    return arr

# O(2^n) - 指数时间
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)
```

## 2. 空间复杂度

### 什么是空间复杂度

空间复杂度表示算法所需存储空间随输入规模增长的变化趋势。

### 常见空间复杂度

| 复杂度 | 说明 | 示例 |
|--------|------|------|
| O(1) | 常数空间 | 原地排序 |
| O(n) | 线性空间 | 创建新数组 |
| O(n²) | 平方空间 | 二维数组 |
| O(log n) | 对数空间 | 递归栈(二分) |

### 示例分析

```python
# O(1) - 常数空间
def sum_array(arr):
    total = 0
    for num in arr:
        total += num
    return total

# O(n) - 线性空间
def create_copy(arr):
    new_arr = []
    for num in arr:
        new_arr.append(num)
    return new_arr

# O(n) - 递归栈空间
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n-1)

# O(n²) - 平方空间
def create_matrix(n):
    return [[0 for _ in range(n)] for _ in range(n)]

# O(log n) - 递归栈空间
def binary_search_recursive(arr, target, left, right):
    if left > right:
        return -1
    mid = (left + right) // 2
    if arr[mid] == target:
        return mid
    elif arr[mid] < target:
        return binary_search_recursive(arr, target, mid+1, right)
    else:
        return binary_search_recursive(arr, target, left, mid-1)
```

### 时间与空间的权衡

```python
# 示例1：查找重复元素

# 方案1：时间O(n²)，空间O(1)
def find_duplicates_brute(arr):
    duplicates = []
    for i in range(len(arr)):
        for j in range(i+1, len(arr)):
            if arr[i] == arr[j] and arr[i] not in duplicates:
                duplicates.append(arr[i])
    return duplicates

# 方案2：时间O(n)，空间O(n)
def find_duplicates_hash(arr):
    seen = set()
    duplicates = set()
    for num in arr:
        if num in seen:
            duplicates.add(num)
        seen.add(num)
    return list(duplicates)
```

## 3. 大O符号详解

### 大O的含义

```
O(f(n)) 表示存在正常数 c 和 n₀，使得对于所有 n ≥ n₀，
算法的运行时间 T(n) ≤ c * f(n)
```

### 常见规则

```python
# 规则1：常数项可以忽略
# O(2n) = O(n)
# O(100) = O(1)

# 规则2：低阶项可以忽略
# O(n² + n) = O(n²)
# O(n³ + n² + n) = O(n³)

# 规则3：系数可以忽略
# O(3n²) = O(n²)
# O(5n log n) = O(n log n)

# 规则4：对数的底可以忽略
# O(log₂n) = O(log₁₀n) = O(log n)
```

### 最好、最坏、平均复杂度

```python
def linear_search(arr, target):
    for i, num in enumerate(arr):
        if num == target:
            return i
    return -1

# 最好情况: O(1) - 第一个元素就是目标
# 最坏情况: O(n) - 目标在最后或不存在
# 平均情况: O(n/2) = O(n)

def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

# 最好情况: O(1) - 中间元素就是目标
# 最坏情况: O(log n) - 目标在最边缘或不存在
# 平均情况: O(log n)
```

## 4. 常见算法

### 排序算法

```python
# 冒泡排序 - O(n²), 稳定
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n-i-1):
            if arr[j] > arr[j+1]:
                arr[j], arr[j+1] = arr[j+1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr

# 选择排序 - O(n²), 不稳定
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        min_idx = i
        for j in range(i+1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
    return arr

# 插入排序 - O(n²), 稳定
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        while j >= 0 and arr[j] > key:
            arr[j+1] = arr[j]
            j -= 1
        arr[j+1] = key
    return arr

# 快速排序 - O(n log n), 不稳定
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    return quick_sort(left) + middle + quick_sort(right)

# 归并排序 - O(n log n), 稳定
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    result.extend(left[i:])
    result.extend(right[j:])
    return result
```

### 搜索算法

```python
# 二分查找 - O(log n)
def binary_search(arr, target):
    left, right = 0, len(arr) - 1
    while left <= right:
        mid = (left + right) // 2
        if arr[mid] == target:
            return mid
        elif arr[mid] < target:
            left = mid + 1
        else:
            right = mid - 1
    return -1

# 深度优先搜索 - DFS
def dfs(graph, start, visited=None):
    if visited is None:
        visited = set()
    visited.add(start)
    print(start, end=' ')
    for neighbor in graph[start]:
        if neighbor not in visited:
            dfs(graph, neighbor, visited)
    return visited

# 广度优先搜索 - BFS
from collections import deque

def bfs(graph, start):
    visited = set()
    queue = deque([start])
    visited.add(start)

    while queue:
        vertex = queue.popleft()
        print(vertex, end=' ')
        for neighbor in graph[vertex]:
            if neighbor not in visited:
                visited.add(neighbor)
                queue.append(neighbor)
```

### 递归

```python
# 阶乘
def factorial(n):
    if n <= 1:
        return 1
    return n * factorial(n-1)

# 斐波那契数列(带缓存)
from functools import lru_cache

@lru_cache(maxsize=None)
def fibonacci(n):
    if n <= 1:
        return n
    return fibonacci(n-1) + fibonacci(n-2)

# 汉诺塔
def hanoi(n, source, auxiliary, target):
    if n == 1:
        print(f"Move disk 1 from {source} to {target}")
        return
    hanoi(n-1, source, target, auxiliary)
    print(f"Move disk {n} from {source} to {target}")
    hanoi(n-1, auxiliary, source, target)
```

## 5. 复杂度分析技巧

### 循环复杂度分析

```python
# 单层循环 - O(n)
def sum_n(n):
    total = 0
    for i in range(n):
        total += i
    return total

# 嵌套循环 - O(n²)
def print_pairs(n):
    for i in range(n):
        for j in range(n):
            print(i, j)

# 顺序循环 - O(n)
def two_loops(n):
    for i in range(n):
        print(i)
    for j in range(n):
        print(j)

# 循环变量倍增 - O(log n)
def halving(n):
    count = 0
    while n > 0:
        n = n // 2
        count += 1
    return count
```

### 递归复杂度分析

```python
# 线性递归 - O(n)
def linear_recursion(n):
    if n <= 0:
        return 0
    return n + linear_recursion(n-1)

# 二分递归 - O(2^n)
def binary_recursion(n):
    if n <= 1:
        return n
    return binary_recursion(n-1) + binary_recursion(n-2)

# 主定理: T(n) = aT(n/b) + O(n^d)
# a: 子问题数量
# b: 子问题规模缩减倍数
# d: 分治以外的工作复杂度指数
```
