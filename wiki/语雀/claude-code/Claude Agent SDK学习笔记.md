## 一、Claude Agent SDK简介

在 Python 项目里以编程方式使用 Claude Code/Claude Agent 的能力，构建可以读文件、改代码、运行命令、搜索、调用自定义工具的 AI Agent。

官方文档说明，Agent SDK 提供了与 Claude Code 相同的工具、Agent Loop 和上下文管理能力，并且可通过 Python 和 TypeScript 编程使用。
