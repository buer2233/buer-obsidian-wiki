# 接口安全测试

## 安全测试方案

### 1. 输入验证测试

| 测试项 | 说明 | 示例 |
|--------|------|------|
| SQL注入 | 通过输入恶意SQL语句 | `' OR 1=1 --` |
| XSS注入 | 注入恶意脚本代码 | `<script>alert(1)</script>` |
| 命令注入 | 执行系统命令 | `; ls -la` |
| 路径遍历 | 访问敏感文件 | `../../etc/passwd` |

### 2. 认证授权测试

- **未授权访问**：直接访问需认证的接口
- **越权访问**：使用低权限用户访问高权限资源
- **Token安全**：Token过期、篡改、重放测试
- **会话管理**：Session固定、会话劫持

### 3. 加密传输测试

- HTTPS证书有效性验证
- 敏感数据传输加密检查
- 密码存储加密方式验证
- API密钥暴露风险检查

### 4. 异常处理测试

```python
# 测试异常输入处理
test_cases = [
    {"input": "", "desc": "空输入"},
    {"input": "A" * 10000, "desc": "超长输入"},
    {"input": None, "desc": "空值"},
    {"input": {"key": "value" * 100}, "desc": "嵌套过深"},
]
```

### 5. 监控日志测试

- 敏感操作是否记录日志
- 日志中是否泄露敏感信息
- 异常访问是否有告警机制

## DVWA 演练网站

**Damn Vulnerable Web Application (DVWA)** 是一个用于安全测试练习的PHP应用。

### 安装步骤

```bash
# 使用Docker快速部署
docker run --rm -it -p 80:80 vulnerables/web-dvwa

# 访问 http://localhost:80
# 默认账号: admin / password
```

### 安全等级

| 等级 | 说明 |
|------|------|
| Low | 无任何安全防护 |
| Medium | 基本过滤 |
| High | 较强防护 |
| Impossible | 安全编码示例 |

### 练习模块

1. **Brute Force** - 暴力破解
2. **Command Injection** - 命令注入
3. **CSRF** - 跨站请求伪造
4. **File Inclusion** - 文件包含
5. **SQL Injection** - SQL注入
6. **XSS** - 跨站脚本攻击

## BurpSuite 基础使用

### 1. 配置代理

```
浏览器代理设置: 127.0.0.1:8080
BurpSuite -> Proxy -> Options -> 确认监听端口
```

### 2. 拦截请求

```
Proxy -> Intercept -> Intercept is on
浏览器访问目标网站 -> 请求被拦截
Forward: 放行  Drop: 丢弃  Action: 发送到其他模块
```

### 3. 常用模块

| 模块 | 功能 |
|------|------|
| Target | 目标站点地图 |
| Proxy | 代理拦截 |
| Intruder | 自动化攻击(爆破、Fuzzing) |
| Repeater | 手动重放请求 |
| Scanner | 漏洞扫描(仅Pro版) |
