## 1. 从列表中随机取两个值

```python
import random
fieldId_list = random.choices(field_df["fieldId"].tolist(), k=2)
```

`random.choices(population, weights=None, cum_weights=None, k=1)`
- population：集群
- weights：相对权重
- cum_weights：累加权重
- k：取值个数

## 17. pytest 8.x 版本不兼容 conftest 写法

**问题表现：** 从 Python 3.7 升级到 3.12（pytest 7.0 → 8.0）后，无法获取根目录 conftest.py 中的全局 fixture。

**问题原因：** pytest 8.x 的 conftest.py 加载规则变更 — 当子目录存在非空 conftest.py 时，会终止向上递归查找更高层级的 conftest.py。

**解决方案：** 在子目录 conftest.py 文件最上方强制引用上层 conftest.py 内的内容。
